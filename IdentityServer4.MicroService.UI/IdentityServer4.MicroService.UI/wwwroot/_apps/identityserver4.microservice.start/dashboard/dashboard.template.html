开发应用

  • 1，安装脚手架
    npm install -g identityserver4.Micoservice.cli //安装脚手架工具
  • 2，创建应用
    ids4ms newapp helloworld //使用脚手架创建1个应用，helloworld名字可以改为自定义名称
  • 3，打包应用
    cd helloworld //进入helloworld文件夹
    npm install //安装应用所需的npm包
    gulp build //打包应用，打包后的目录为build文件夹
  • 4，发布应用
    cd build //进入build文件夹
    npm publish //发布包到npm平台
  • 5，发布到平台
    cd ../ //返回应用所在根目录
    ids4ms publish //发布包到服务器

    应用根目录ids4ms.config.js配置说明

    server：后台地址，默认为（https://localhost:44386/Home/RegisterApp）

    name：应用名称

    icon：应用图标

    token：发布令牌
