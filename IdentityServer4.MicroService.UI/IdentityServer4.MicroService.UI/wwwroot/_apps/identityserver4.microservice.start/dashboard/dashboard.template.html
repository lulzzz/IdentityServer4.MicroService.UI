开发应用

  • 1，安装脚手架
    npm i -g identityserver4.microservice.cli //安装脚手架工具
  • 2，创建应用
    ids4ms newapp helloworld //使用脚手架创建1个应用，helloworld名字可以改为自定义名称
  • 3，本地发开&嵌入平台调试
    cd helloworld //进入helloworld文件夹
    npm install //安装应用所需的npm包
    npm run start //启动应用，默认网址http://127.0.0.1:8080
    嵌入平台调试：打开{{devUrl}}?dev=http://127.0.0.1:8080 {http://127.0.01:8080是您的应用url}
  • 4，打包应用
    gulp build //打包应用，打包后的目录为build文件夹
  • 5，发布应用
    cd build //进入build文件夹
    npm publish //发布包到npm平台
